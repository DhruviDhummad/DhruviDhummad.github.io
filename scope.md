# Scope of Assessment

## In Scope
- Target application: app.talentin.ai
- Authenticated and unauthenticated user flows
- Passive and low-impact security testing
- Review of:
  - HTTP security headers
  - Authentication and session handling
  - Basic input validation checks

## Out of Scope
- Denial of Service (DoS) testing
- Brute-force attacks
- Data destruction or modification
- Aggressive automated exploitation
- Testing of third-party services

## Rules of Engagement
- Testing performed with authorization
- No disruption to production services
- Findings documented with evidence only
